# Poem_operator
This is a  menu driven program to    
•    1. Count no of dots '.'   
•    2. Count word ‘I’   
•    3. Display lines starting with ‘tT’   
•    4. Display the first &amp; last line    
•    5. Display the longest line 

Concept Used :  Text Files

Can use it with a .txt file in the same directory .
